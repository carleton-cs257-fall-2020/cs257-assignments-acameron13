NAME
	books.py

SYNOPSIS
	python3 books.py -a=[SEARCHSTRING]
	python3 books.py -d --from=[STARTYEAR] --to=[ENDYEAR]
	python3 books.py -h
	python3 books.py -t=[SEARCHSTRING]

DESCRIPTION
	books.py can perform various operations on a dataset of books, given the following mandatory command-line arguments. Books.py has the ability to display a list of books within a specified inclusive range of dates. It can also display all books whose titles contain a given search string, as well as display all authors whose names contain a given search string, as well as all books published by those authors. Only one of these operations can be performed at one time.

OPTIONS
	-a=[SEARCHSTRING]      --author=[SEARCHSTRING]
		Displays all authors whose names contain the given SEARCHSTRING, as well as all of their books. The SEARCHSTRING is case insensitive. The SEARCHSTRING is mandatory for this operation.

	-d --from=[STARTYEAR] --to=[ENDYEAR]     --daterange --from=[STARTYEAR] --to=[ENDYEAR]
		Displays all books that were published within the given inclusive date range. STARTYEAR and ENDYEAR are mandatory for this operation.

  	-h     --help
	  	Display this help page and exit

  	-t=[SEARCHSTRING]       --title=[SEARCHSTRING]
	  	Displays all books where the title contains the given SEARCHSTRING. The SEARCHSTRING is case insensitive. The SEARCHSTRING is mandatory for this operation.
